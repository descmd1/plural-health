# Healthcare Patient Management System - Technical Assessment

A responsive patient management dashboard built with **Next.js 14**, **Tailwind CSS**, and **Lucide React** as part of a technical assessment.

## Project Overview

This project demonstrates the implementation of a modern healthcare patient management interface with the following key requirements:

- **Patient Data Table** with sorting, filtering, and responsive design
- **Advanced Search** functionality across multiple patient fields
- **Modal Interfaces** for appointment creation and patient management
- **Responsive Design** that works seamlessly on desktop and mobile
- **Interactive UI Components** with modern user experience patterns

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Language**: JavaScript/JSX

## Implemented Features

### Core Functionality
-  Patient data table with realistic healthcare data
-  Advanced search with multi-field filtering (name, patient ID, clinic, age, gender)
-  Responsive design - table transforms to cards on mobile
-  Interactive modals for appointment creation
-  Real-time search with dropdown results
-  Status tracking with visual indicators

### UI/UX Features
-  Fixed header with scrolling table body
-  Mobile hamburger menu with user options
-  Desktop user dropdown menu
-  Hover states and smooth transitions
-  Filter toggles with active filter count
-  Loading states and empty states

### Technical Implementation
-  Component-based architecture
-  Responsive grid and flex layouts
-  Custom hook patterns for state management
-  Reusable modal components
-  Proper accessibility attributes
